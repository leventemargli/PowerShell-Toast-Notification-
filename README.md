# PowerShell Toast Notification - ???

A custom Windows 10 Metro-style notification script that replicates system-level battery alerts with a glitched twist.

## Features
* **Background Dimming:** Uses a full-screen shadow layer to focus the user on the message.
* **Custom UI:** Replicates the blue Windows 10 dialog box exactly.
* **Glitched Text:** Utilizes Unicode Zalgo text for a "creepy" aesthetic.

## ⚠️ Warning
The **Shutdown** button is functional. It will trigger a forced system shutdown. Use for educational/demonstration purposes only.

## How to Run
1. Download `unknown.ps1`.
2. Ensure you save it with **UTF-8 with BOM** encoding.
3. Right-click and select **Run with PowerShell**.
